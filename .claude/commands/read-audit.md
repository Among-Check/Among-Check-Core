Read the version-controlled security audit before making changes.

1. Open `audits/latest.toon` — current findings
2. Open the newest `audits/history/*/delta.toon` — new vs resolved since last run
3. Summarize: critical/high count, top 3 imposters, suggested fix order
4. Reference `docs/audit-archive.md` for TOON field meanings

If `latest.toon` shows `scanId: pending`, note that no scan has run yet.
